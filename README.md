# RTFM Capital Bank

A full-stack banking application built with the MERN stack (MongoDB, Express.js, React, Node.js).

## Features

- User Authentication (Register/Login)
- Multiple Account Management
- Money Transfers between accounts
- Deposit and Withdrawal functionality
- Transaction History
- Real-time balance updates
- Responsive dashboard with charts
- Profile Management

## About RTFM Capital Bank

RTFM Capital Bank provides modern, secure, and user-friendly banking services. Our platform demonstrates the power of the MERN stack in creating robust financial applications.


## Prerequisites

- Node.js (v14 or higher)
- MongoDB (local installation or MongoDB Atlas)
- npm or yarn package manager
